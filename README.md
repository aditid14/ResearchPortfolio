# Aditi Deshpande – Computational Biology / Quantitative Research Portfolio

This repository contains selected coursework and research projects in computational biology, bioinformatics, and quantitative modeling. These materials highlight my experience in analyzing high-dimensional biological datasets, implementing computational workflows, and applying quantitative methods to cancer research.  

## Contents

### Research Projects
1. **SURF Breast Cancer Project** (`SURF.ipynb`)  
   - Summer Undergraduate Research Fellowship (SURF) project at the Purvis Lab, UNC Chapel Hill.  
   - Stratification of resistant breast cancer cell populations using clustering algorithms and machine learning techniques, including graph neural networks.  

2. **Tumor-Normal Differential Transcriptomics** (`Differential Transcriptomics and Pathway Analysis of Tumor versus Normal Colorectal Tissue (4).pdf`)  
   - Functional Genomics class project.  
   - Differential transcriptomics and pathway analysis comparing tumor versus normal colorectal tissue using DESeq2 and pathway enrichment methods.  

3. **Tumor-Normal-Immune Mathematical Modeling** (`Tumor-Normal-Immune Math Modeling (1).pdf`)  
   - UNC Direct Research Program project.  
   - Mathematical modeling of tumor progression and immune response using differential equations.  

### Code Files
- **`DRP_treatment.m`** – MATLAB code for the tumor modeling project.  
- **`SURF.ipynb`** – Jupyter Notebook for SURF computational analyses, including clustering and preprocessing pipelines.  

## Skills Demonstrated
- Programming in Python, R, and MATLAB  
- Data analysis and statistical modeling  
- High-dimensional biological data processing (DESeq2, pathway enrichment)  
- Machine learning (clustering algorithms, graph neural networks)  
- Computational workflows and reproducible research  

## How to View / Run
- To view notebooks, open `.ipynb` files in JupyterLab or Jupyter Notebook.  
- To run MATLAB scripts, open `.m` files in MATLAB.
